Loading up a Mod Running on the Source Engine

While visiting an old project, Space Escape, I realized that I had forgoten how to load it into Steam and launch it. So here are the steps that I took:

  1. Make sure to have Half-Life 2 and Half-Life 2: Deathmatch installed as well as the Steam tools, found under Library | Tools, Source SDK and Source SDK Base 2006.
  2. Locate the mod folder that goes into the sourcemods folder in Steam and place it in there (..Steam\steamapps\sourcemods).
  3. If there is one, place the other folder in the root directory where Steam is located. For me this was D:\, but for most people this will be C:\.
    1. If there isn't a second folder, launch Source SDK.
    2. Set the Engine Version to Source Engine 2006.
    3. Set the Current Game to Half-Life 2.
    4. Launch Create a Mod by double-clicking it.
    5. Modify Half-Life 2 Singleplayer.
    6. Set the first location to C:\McMod, create a folder if necessary.
    7. Set the name of the mod to mcmod.
    8. Click Next.
    9. Now the files should have been created and you can exit Source SDK.
  4. Restart Steam.
  5. The mod should now be located under the My Games tab in your steam library of Games.
  6. After this, load up the game, and to load the map of Space Escape open the console (~), and type "map mod_gold" and hit enter.
